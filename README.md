This project analyzes pizza store sales data using PostgreSQL to uncover valuable business insights such as top-selling pizzas, customer ordering trends, revenue generation, peak order timings, and category-wise performance. 

The project includes data importing, table relationships, SQL joins, aggregation queries, and business-focused analysis to support data-driven decision making in the food industry.

Key skills used:
• PostgreSQL
• SQL Joins
• Aggregate Functions
• Data Cleaning
• Business Analysis
• Revenue & Sales Insights

The project is designed to simulate a real-world business case and strengthen practical SQL and analytical skills.
